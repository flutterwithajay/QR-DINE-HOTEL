QR DINE HOTEL

**Scan QR → Order Food → Kitchen Gets Real-Time Orders**

TableDine is a complete digital dining solution for hotels and restaurants. Guests scan a QR code on their table to access the menu and place orders. Kitchen staff receives orders instantly on a dedicated mobile app.

## ✨ Features

### Guest Web App
- 🚫 No app download - works in any browser
- 🍕 Digital menu with categories
- 🛒 Add to cart with real-time total
- ✅ Instant order confirmation
- 📊 Track order status

### Kitchen App
- 🔔 Real-time order notifications
- 📋 View and manage orders
- 🏷️ Update status (Pending → Preparing → Ready → Served)
- 📱 Mobile-first design
